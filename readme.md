### 对比监督学习和自监督学习在图像分类任务上的性能表现
---------------------------------------------------

#### 介绍
  使用ResNet-18作为基础模型,选用simclr对比学习算法在cub数据集中训练模型，并在cifar数据集上进行检验。对比监督学习和自监督学习在图像分类任务上的性能表现。

#### 实现
  可运行final.ipynb文件，cub数据集见CUB_200_2011.zip文档
